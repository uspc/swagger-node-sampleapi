Prerequest 
- Install postgresql

To try out
$ git clone https://github.com/uspc/swagger-node-sampleapi.git
$ npm install
$ psql -f puppies.sql
$ npm run start

Browse swagger json
http://localhost:3000/swagger.json

Browse swagger UI 
http://localhost:3000/api-docs/#!/Puppies/get_api_puppies

Sample app url
http://localhost:3000/api/puppies
